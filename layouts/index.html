SQL Sources

sq supports input from and output to common databases.

  • Postgres
  • SQLite
  • SQL Server / Azure SQL Edge
  • MySQL

Other Sources

In addition to SQL sources, sq can import data from these formats:

  • Excel XLSX
  • CSV
  • TSV
  • JSON (beta)
  • JSONL (JSON Lines)
  • JSONA (JSON Array)
  • XML

Output Formats

sq can output to a plethora of file formats, or insert results directly to a database table.

  • SQL INSERT to Postgres, SQLite, SQL Server / Azure SQL Edge, MySQL
  • Excel XLSX
  • CSV
  • TSV
  • JSON
  • JSONL (JSON Lines)
  • JSONA (JSON Array)
  • XML
  • HTML
  • Markdown
  • Raw (bytes)
