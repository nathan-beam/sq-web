List available drivers.

Usage:
  sq driver ls [flags]

Flags:
  -j, --json     Output JSON
  -t, --table    Output text table
  -h, --header   Print header row in output (default true)
      --help     help for ls

Global Flags:
  -M, --monochrome   Don't colorize output
  -v, --verbose      Print verbose output, if applicable
