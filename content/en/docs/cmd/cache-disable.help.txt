Disable caching. This is equivalent to:

  $ sq config set ingest.cache false

Usage:
  sq cache disable

Examples:
  $ sq cache disable

Flags:
      --help   help for disable

Global Flags:
      --config string      Load config from here
      --log                Enable logging
      --log.file string    Path to log file; empty disables logging
      --log.level string   Log level: one of DEBUG, INFO, WARN, ERROR
  -M, --monochrome         Don't colorize output
      --no-progress        Progress bar shown for long-running operations
  -v, --verbose            Verbose output
